BASIC ECO Pick Fill and Seal (PFS) machine

  • home

These machines use four sides sealed / stand up preformed pouches which may be pre-fitted with corner spouts or zippers.

This is a single station pick fill and seal machine comprising

  1. Magazine for pouch loading
  2. Reciprocating arm for pouch transfer for filling.
  3. Single station to receive, open, fill, seal and release the pouch.
  1. Conveyor for pouch output
  2. Filler according to the product to be packed.
  3. Drive and controls for pneumatics are controlled by PLC in a panel.

Machine Features

  1. The machine is economical and compact.
  2. Packs as much as 2 kg/litre in a pouch.
  3. A vertically oscillating chute/nozzle conveys the fill quantity of the product into the pouch so as to avoid product contamination in seal area.
  4. Facility for data storage and downloading to PC, for production statistics, is provided.
  1. In this machine, one can change the pouch size (width), in the given range of the machine specification, simply, by changing the machine settings.
  2. Machine interlocks, such as “no bag no fill” and “no product no bag feed”, are provided to avoid wastage and downtime for cleaning. The operator is signalled when the interlocks are actuated.
